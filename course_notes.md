### Jupyter keyboard shortcuts

- Enter Command mode: ESC
- Enter Edit mode: ENTER
- Command mode keys
  - insert cell below: b
  - insert cell above: a
  - delete cell: dd
  - convert cell to MarkDown: m
  - convert cell to code: y
  - Cell to heading 1-6: 1-6
  - Cell to raw: r
  - Cut cell: x
  - Copy cell: c
  - Paste cells below: v
  - Paste cells above: SHIFT+v
